# Your Website

A modern, responsive website built with HTML, CSS, and JavaScript.

## Features

- ✨ Responsive design (mobile, tablet, desktop)
- 🎨 Modern UI with smooth animations
- 📱 Sticky navigation bar
- 📧 Contact form
- ⚡ Fast and lightweight

## Sections

1. **Hero** - Eye-catching banner with CTA button
2. **About** - Your story and mission
3. **Services** - What you offer
4. **Contact** - Get in touch form
5. **Footer** - Copyright info

## How to Use

1. Download or clone the files
2. Open `index.html` in your browser
3. Customize the content:
   - Update brand name in navbar
   - Change colors in `styles.css` (`:root` variables)
   - Edit text and add your own content
   - Update service cards
   - Modify contact form

## Customization

### Change Colors
Edit the `:root` variables in `styles.css`:
```css
--primary-color: #007bff;
--secondary-color: #0056b3;
```

### Add More Sections
Copy and paste a section block in `index.html` and style as needed.

### Deploy
Upload files to any web hosting or GitHub Pages:
1. Push to GitHub
2. Enable GitHub Pages in repository settings
3. Your site is live!

## Browser Support

- Chrome, Firefox, Safari, Edge (latest versions)
- Mobile browsers

Enjoy! 🚀
